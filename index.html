CYNIDZO

by Esone Xon


Aplikasi android gratis yang dibuat dengan tujuan mengedukasi seputar Dunia Cyber dan Teknologi.

Spesifikasi Aplikasi

Aplikasi dibuat dengan pemrograman Java dan Kotlin. Dibuat minimal untuk Android 5. Bahasa yang digunakan Indonesia

Fitur Aplikasi

Aplikasi ini dibuat hasil kegabutan, jadi Fitur yang tersedia pun random dan pastinya berfungsi, fitur yang baru dibuat ialah Tutorial, Script, E-Book, Tools(Downloader), Code Editor, Dark Web Link Search, dll

Fitur Spesial

Aplikasi ini bisa membuka OS(Operating Sistem).

Aman Dari Virus
